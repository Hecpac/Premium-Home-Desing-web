# Design System Spec: "Premium White Corsa" (v3.0)

## 1. Filosofía Visual
- **Base:** "High Key Architecture". Fondos blancos, luz abundante, sombras difusas.
- **Estilo:** Minimalismo Suizo. El lujo se define por el espacio en blanco, no por la decoración.
- **Uso del Rojo:** CRÍTICO. Solo el 5% de la UI (Botones primarios y punteros).
- **Grid:** Visible mediante líneas finas (`border-gray-200`), no por bloques de fondo.

## 2. Paleta de Colores (Light Mode)
- **Canvas:** `#FFFFFF` (Blanco puro).
- **Surface:** `#FAFAFA` (Gris casi blanco para diferenciar secciones sutilmente).
- **Text Primary:** `#18181B` (Zinc-900, Negro Tinta).
- **Text Secondary:** `#71717A` (Zinc-500, Gris Técnico).
- **Accent:** `#D40000` (Rosso Corsa). Usar con extrema moderación.

## 3. Tipografía & Layout
- **Heading:** Geist Sans Black (Heavy weight). Color: Negro Tinta.
- **Body:** Geist Sans Light/Regular.
- **Estructura:** Bento Grid definido por BORDES, sin sombras pesadas.
